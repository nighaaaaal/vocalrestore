# 🎙️ VocalRestore: The "You Thought You Were Safe" Audio Reverser 🕵️‍♂️

**VocalRestore** is a web-based audio manipulation tool designed to completely reverse social media voice filters.

We all have that *one* friend. The one who confidently sends a WhatsApp or Instagram voice note of themselves singing terribly, but they hide behind the "Chipmunk" or "Demon" voice filter thinking their dignity is mathematically protected by the algorithm.

They thought wrong. 😈

I built this tool to act as a digital forensic laboratory. It mathematically counteracts pitch-shifting and time-stretching so you can recover their **real, unfiltered voice**, export it as an MP3, and send it right back into the group chat.

## ✨ Features

* 🎭 **One-Click Forensics (Presets):** Instantly apply counter-measures for popular filters like Chipmunk, Demon, Princess, and Grandpa.
* 🎚️ **Manual Interrogation (Pro Sliders):** Every vocal tract is different. Fine-tune the Pitch, Playback Speed, and High EQ to perfectly unmask your victim's true voice.
* 💾 **Instant Weaponization (MP3 Export):** Automatically mathematically encodes the restored audio into a lightweight, WhatsApp-ready `.mp3` file.
* 🛡️ **Zero-Trust Security:** Your friends' terrible singing never touches a server. Everything is processed **100% locally** in your browser's memory using the Web Audio API.

## 🛠️ The Tech Stack

No bloated backend frameworks here. Just pure, client-side wizardry:

* **[Tone.js](https://tonejs.github.io/):** The core audio engine handling real-time pitch shifting and equalization context bridging.
* **[lamejs](https://github.com/zhuker/lamejs):** An open-source MP3 encoder running entirely inside the browser to ensure WhatsApp compatibility.
* **[Tailwind CSS](https://tailwindcss.com/):** For a sleek, dark-mode, glassmorphic UI. (Because if you're going to expose your friends, you should look like a pro audio engineer doing it).
* **[Phosphor Icons](https://phosphoricons.com/):** Clean, modern iconography.

## 🚀 How to Use (Quick Start)

Because there is no backend server, deploying and running this app is insanely easy.

1. Clone this repository:
   ```bash
   git clone [https://github.com/nighaaaaal/vocalrestore.git](https://github.com/nighaaaaal/vocalrestore.git)
   ```
2. Open the folder and double-click `index.html` to open it in any modern web browser.
3. Upload a screen recording (`.mp4`) or audio file (`.mp3`, `.wav`) of the filtered voice note.
4. Hit a preset, adjust the sliders until they sound ridiculous, and hit **Export MP3**.

*Note: For the absolute best results, use the manual sliders! Pre-set filters are a great starting point, but manual adjustments give the most accurate "gotcha" moments.*
